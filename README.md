#BMI 計算機

一個身體質量指數計算工具，使用者可輸入身高與體重，快速計算 BMI，並顯示分類與健康建議。

## 功能特色

* 輸入身高（cm）與體重（kg）
* 即時計算 BMI
* 顯示以下資訊：
  * BMI 數值（四捨五入至小數點後兩位）
  * BMI 分類
  * 健康建議


## 輸入驗證

* 若未完整輸入身高或體重：

  * 系統會顯示提示訊息
    「請輸入完整的身高與體重」

##  BMI 分類標準

| BMI 範圍      | 分類 |
| ----------- | -- |
| < 18.5      | 過輕 |
| 18.5 – 24.9 | 正常 |
| 25 – 29.9   | 過重 |
| ≥ 30        | 肥胖 |

* 按下 **Enter 鍵**即可快速進行 BMI 計算
